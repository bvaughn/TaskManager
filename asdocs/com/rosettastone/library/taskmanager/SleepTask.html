Classpublic class SleepTask
InheritanceSleepTask InterruptibleTask Task flash.events.EventDispatcher

Sleeps until told to complete. This Task can be inserted into a composite to block for an indeterminate amount of time. Since this Task is simple a placeholder, it supports interruptibility.
